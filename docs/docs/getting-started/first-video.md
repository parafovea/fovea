# Your First Video

A detailed tutorial on loading a video, creating annotations, and exporting data.

## Load Your Video

1. Navigate to the video management section
2. Click **Upload Video**
3. Select a video file (MP4, WebM, or OGG format)
4. Wait for upload and processing to complete

## Create Annotations

1. Select the video from the list
2. Use the annotation toolbar to:
   - Draw bounding boxes
   - Create temporal annotations
   - Link to world objects

## Export Your Work

1. Click **Export** in the toolbar
2. Choose export format (JSON Lines)
3. Select what to include in the export
4. Download the export file
