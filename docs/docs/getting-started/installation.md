# Installation

This guide will help you install and set up Fovea on your system.

## Prerequisites

- Docker Desktop 4.0 or later
- 8GB RAM minimum (16GB recommended)
- 10GB free disk space

## Quick Install

The fastest way to get Fovea running is with Docker Compose:

```bash
docker compose up
```

This will start all required services including PostgreSQL, Redis, and the Fovea application.
